# Movie-and-OTT-Platform-Dashboard

## 📌 Project Overview

The **Movie OTT Platform Analysis Dashboard** is a data analytics project developed to analyze movies and OTT platform trends using interactive visualizations. The dashboard helps users understand movie performance, genre popularity, ratings, revenue, and OTT platform comparisons through meaningful insights.

This project was created using data analysis and visualization tools to improve decision-making and understand audience preferences in the entertainment industry.

---

## 🚀 Features

* Analyze movie ratings and reviews
* Compare OTT platforms based on movie collections
* Genre-wise movie distribution analysis
* Year-wise release trends
* Top-rated and most popular movies
* Interactive filters and charts
* User-friendly dashboard interface

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Power BI / Tableau
* Excel
* SQL

---

## 📊 Dashboard Insights

* Identified the most popular OTT platform based on movie ratings
* Analyzed trending genres among audiences
* Compared yearly movie releases across platforms
* Found relationships between ratings, votes, and popularity

---

## 📂 Dataset

The dataset contains information about:

* Movie titles
* Genres
* Ratings
* Release year
* OTT platform names
* Popularity scores
* Votes and reviews

---

## 🎯 Objective

The main objective of this project is to perform data analysis on OTT platform movie data and present insights visually to help understand audience behavior and entertainment trends.
